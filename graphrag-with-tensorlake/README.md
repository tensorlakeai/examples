# GraphRAG with Tensorlake

This example demonstrates how to use Tensorlake with a GraphRAG-style workflow.
It is adapted from [Studio1HQ/GraphRAG-with-TensorLake](https://github.com/Studio1HQ/GraphRAG-with-TensorLake/tree/main).

## Setup

```bash
pip install -r requirements.txt
python tensorlake_graphrag.py
```
